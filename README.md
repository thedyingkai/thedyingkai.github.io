# thedyingkai.github.io

个人博客新站。当前正在从旧站 `https://blog.thedyingkai.cn/` 迁移内容。

## 当前状态

- 首页、文章归档、项目页、关于页、404、favicon、sitemap、robots 已搭好。
- 之前生成的占位博客内容已从线上入口撤下。
- 等旧站内容可抓取后，把旧站文章迁移为新站归档。

## 部署方式

在仓库 Settings -> Pages 中选择：

- Source: Deploy from a branch
- Branch: main
- Folder: /root

之后访问：

```txt
https://thedyingkai.github.io/
```
