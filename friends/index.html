Pull Request

你可以这样将你的友链填写到我的网站

  1. Fork 仓库

    点击 GitHub 仓库右上角的 Fork 按钮，把整个项目复制到你自己的 GitHub 账号下。

  2. 克隆你的项目副本

    打开终端或 Git Bash，把 YourGitHubID 换成你的 GitHub 用户名。

    git clone https://github.com/YourGitHubID/thedyingkai.github.io.git
cd thedyingkai.github.io

  3. 新建分支

    分支名可以自定义，比如 add-my-name。

    git checkout -b add-my-name

  4. 编辑友链文件

    找到并打开 config/friends.json。在文件的 "links" 数组内部，找到最后一个友链对象，在它后面加逗号，然后追加你的信息。

    "links": [
  {
    "meta": "Friend",
    "title": "站点名称",
    "text": "描述",
    "href": "站点 URL",
    "tags": [
      "Blog"
    ],
    "avatar": "头像 URL"
  }
]

  5. 保存并暂存修改

    git add config/friends.json

  6. 提交并推送分支

    git commit -m "friends: add my blog link"
git push origin add-my-name

  7. 创建 Pull Request

    回到你的项目副本页面：https://github.com/YourGitHubID/thedyingkai.github.io。点击黄色提示旁的 Compare & pull request。

  8. 检查并提交

    在新页面检查友链信息是否正确，然后点击 Create pull request。
