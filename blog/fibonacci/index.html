斐波那契数列

斐波那契性质、求法和齐肯多夫定理，来自旧博客 p=97。

#数学#数论#笔记

定义

$$F_n=\begin{cases}1,&n\le2\\F_{n-1}+F_{n-2},&n>2\end{cases}$$

性质

$$F_n=F_{n-1}+F_{n-2}$$

$$F_n=\frac1{\sqrt5}\left(\frac{1+\sqrt5}{2}\right)^n-\frac1{\sqrt5}\left(\frac{1-\sqrt5}{2}\right)^n$$

$$F_{m+n}=F_mF_{n+1}+F_{m-1}F_n$$

$$\sum_{i=1}^{n}F_i=F_{n+2}-1$$

$$\sum_{i=1}^{n}F_i^2=F_nF_{n+1}$$

$$\sum_{i=1}^{n}F_{2i}=F_{2n+1}-1$$

$$\sum_{i=1}^{n}F_{2i-1}=F_{2n}$$

$$F_n^2-F_{n-1}F_{n+1}=(-1)^{n+1}$$

快速倍增

$$F_{2k}=F_k(2F_{k+1}-F_k),\quad F_{2k+1}=F_{k+1}^2+F_k^2$$

using i64=long long;
pair<i64,i64> fib(i64 n){
  if(n==0) return {0,1};
  auto [x,y]=fib(n>>1);
  i64 c=x*(2*y-x);
  i64 d=x*x+y*y;
  if(n&1) return {d,c+d};
  else return {c,d};
}

和卢卡斯数列的关系

$$L_n=\left(\frac{1+\sqrt5}{2}\right)^n+\left(\frac{1-\sqrt5}{2}\right)^n$$

$$F_{n-1}+F_{n+1}=L_n,\quad L_{n-1}+L_{n+1}=5F_n,\quad F_nL_n=F_{2n}$$

齐肯多夫定理

任何自然数 $n$ 可以被唯一地表示成一些斐波那契数的和，并且不能使用两个相邻的斐波那契数。
