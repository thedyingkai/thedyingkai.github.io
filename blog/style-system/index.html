样式系统先从变量开始

把颜色、圆角、阴影、正文宽度和字体全部抽成 tokens，后续改风格会轻很多。

2026.06.23#css#frontend

这个站的视觉基础不是组件库，而是一组 CSS variables。

这样做的好处是：

  1. 改主题不会散落在每个组件里；
  2. 页面可以保持统一的纸面感、细线、按钮、卡片和排版；
  3. 后续要加暗色模式或者主题切换，只需要覆盖变量。

设计变量

当前首版把页面拆成几类变量：纸面、墨色、细线、强调色、金属色、阴影、字体和正文宽度。

:root {
  --paper: oklch(98.7% .004 252);
  --oxford: oklch(37% .072 252);
  --serif: "Noto Serif SC", "Songti SC", serif;
}

真正高级的个人站，通常不是动效堆得多，而是间距、层级、字体、留白、边线这些细节足够稳定。
